# Largest Element in an Array

## Problem

Find the largest element present in the array.

---

## Approach 1 - Brute Force

### Idea

Sort the array and return the last element.

### Time Complexity

O(N log N)

### Space Complexity

O(1)

---

## Approach 2 - Optimal

### Idea

Traverse the array once while maintaining the maximum element.

### Time Complexity

O(N)

### Space Complexity

O(1)
